# Xcode-Mis-registros-app-

Aplicación desarrollada en SwiftUI para registrar gestos y sus significados.

## Tecnologías
- Swift
- SwiftUI

## Funcionalidades
- Agregar un gesto.
- Agregar un significado.
- Guardar registros en memoria.
- Mostrar registros en pantalla.

## Conceptos de SwiftUI aprendidos
- `@State`
- `@Binding`
- `NavigationStack`
- `NavigationLink`
- `TextField`
- `Button`
- `ForEach`
- Modelos con `struct`
- Protocolo `Identifiable`
- Comunicación entre vistas mediante `@Binding`
